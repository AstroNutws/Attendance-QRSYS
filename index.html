Frequently Asked Questions

What is the difference between Admins? ▼
Standard Admins handle daily operations like monitoring and student finding. Super Admins act as system overseers with the exclusive power to create other admin accounts, update credentials (emails/passwords), and access the specialized Super Admin management panel.
Who has access to the database? ▼
Neither Standard nor Super Admins have direct database access. Ownership is strictly isolated to the Client (Database Owner) via the Supabase Dashboard. This ensures that system-level management remains separated from core data storage for maximum security.
How are passwords secured? ▼
All passwords undergo SHA-256 one-way hashing before transmission. The system never stores or transmits plaintext passwords, ensuring that even in the event of database access, user credentials remain unreadable.
