## Overview
QRSYS is a professional-grade attendance logging system designed for organizations, institutions, and businesses of any scale. It provides a synchronized infrastructure across desktop and mobile platforms, utilizing QR code scanning to efficiently manage and track attendance.

## Features
- **Dual Platform Integration:** Windows Desktop & Android Mobile.
- **Real-time Synchronization:** Powered by Supabase Cloud.
- **Email Notifier:** Automated alerts to designated contacts including registration confirmation emails.
- **Admin Dashboard:** Announcement management and member filtering.
- **Rapid Scanning:** Optimized autofocus for webcam and mobile cameras.
- **Smart Reporting:** Instant CSV exports and daily attendance analytics.

## 🔒 Security & Data Privacy
QRSYS is built with a "Security First" approach to protect all organizational data:

- **Secure Credential Storage:** All passwords are protected using industry-standard hashing before storage. Plain-text passwords are never written to the database.
- **Encrypted Transit:** All data sent between the clients (PC/Mobile) and the cloud is encrypted via SSL/TLS.
- **Minimal Data Footprint:** No biometric data (fingerprints/face scans) is stored. The system only tracks the necessary QR Identifier linked to member profiles.
- **Authorized Access:** Management features are locked behind secure admin credentials to prevent unauthorized data exposure.
- **Role-Based Access Control:** Super Admin privileges restrict sensitive operations to authorized accounts only.

## System Requirements

### Desktop Client
- **OS:** Windows 7 - 11 (x64/x86)
- **Imaging:** USB Webcam (720p+ recommended for speed)
- **Storage:** 400MB Required
- **Network:** Stable Wi-Fi/LAN for Cloud Sync

### Mobile Client
- **OS:** Android 8.0 (Oreo) or higher
- **Hardware:** Camera with Autofocus capability
- **Memory:** 2GB RAM Minimum
- **Capacity:** 50MB Available
