Frequently Asked
FAQ

Common Questions

What is the difference between Admin types?▼
Standard Admins handle daily operations. Super Admins are system overseers with exclusive authority to create accounts, update credentials, and access the Super Admin panel.
What can the Super Admin do?▼
Super Admins register new admins, modify peer details (email, username), reset passwords, and delegate Super Admin privileges via the dedicated Super Admin Panel.
Who has direct database access?▼
Neither admin tier has direct database access. Ownership is isolated exclusively to the Client via their private Supabase Dashboard — system management is fully separated from data storage.
How are credentials protected?▼
Passwords are visually masked in the UI to prevent shoulder-surfing. Future iterations will introduce server-side cryptographic hashing for complete backend protection.
