# Contributing & Feedback for QRSYS

Thank you for using the QRSYS Attendance Logging System. Since this is a proprietary project, we do not accept direct code contributions (Pull Requests) at this time. However, user feedback is vital for the system's growth.

## Reporting Bugs
If you encounter technical issues, "green screen" errors, or synchronization delays, please report them directly via email.

**How to report:**
1. Send an email to: **samongud@gmail.com**
2. Subject: `[BUG] Brief description of the issue`
3. Include:
   - Whether the issue is on the **Desktop** or **Mobile** client.
   - Your Device Model / Windows Version.
   - A screenshot or video of the error if possible.

## Feature Requests
We are constantly looking to improve QRSYS for schools and organizations. If you have a suggestion for a new feature (e.g., specific reporting formats, SMS integration, or UI improvements), please let us know!

**How to suggest:**
1. Send an email to: **samongud@gmail.com**
2. Subject: `[FEATURE] Your Idea Name`
3. Describe how this feature would help your organization's workflow.

## Organizational Support
For setup assistance, custom branding, or deployment inquiries, please use the same contact email above.
